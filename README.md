📖 О проекте / Пра праект
🇷🇺 
Этот Telegram-бот был создан учащимися Витебского государственного технического колледжа из группы ВР-21 для помощи путешественникам и жителям Беларуси. Бот предоставляет подробную информацию о городах, погоде, достопримечательностях и областях страны.

🇧🇾 
Гэты Telegram-бот быў створаны навучэнцамі Віцебскага дзяржаўнага тэхнічнага каледжа з групы ВР-21 для дапамогі падарожнікам і жыхарам Беларусі. Бот прадастаўляе падрабязную інфармацыю пра гарады, надвор'е, славутасці і вобласці краіны.
